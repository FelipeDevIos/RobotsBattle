## README

Relocation based on the Pythagorean theorem


Possible cells:

         ----------
         |        |
         | x, y-1 |
         |        |
----------------------------
|        |        |        | 
| x-1, y |   *    | x+1, y |
|        |        |        |
----------------------------
         |        |
         | x, y+1 |
         |        |
         ----------
